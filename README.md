# Jazykaintegrace — Language School Platform

> **Live: [ji-pepa-final.vercel.app](https://ji-pepa-final.vercel.app)**

Full-stack web application for a Czech language school — course catalog, student registration, and online payments.

## Tech Stack

- **React 18** + **TypeScript** + **Vite**
- **Tailwind CSS** — custom responsive design
- **Supabase** — auth, database, storage
- **Stripe** — course payments and subscriptions
- **Vercel** — deployment

## Features

- Course catalog with filtering and search
- Student registration and booking flow
- Stripe payment integration
- Teacher profiles and schedule management
- Blog / news section
- SEO optimized, fully responsive

## Development

```bash
npm install
npm run dev
```
